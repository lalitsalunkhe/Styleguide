Typography

Serif for the moments — greetings, AI message bodies, display headlines. Sans for every UI surface and chrome. Mono is reserved for code and technical identifiers. Lose the serif and it stops looking like Claude. Displayed using the active font pairing — toggle from the top bar to see the alternate.

Conditional formatting in tables

When the cell is the data point, colour it — don't paint the whole row. 2px gaps between cells make the heatmap read as discrete swatches rather than noise.

Accordions

No outer card. Hairline dividers, chevron rotates 180 degrees, body height eases via the grid 0fr → 1fr trick.

A learning surface for the visual language we're building on top of Claude's design system.
A sliding indicator preserves spatial continuity — the eye tracks where it just was. Fades break that mental map.
Borders carry elevation; shadows pull back. Brown stays brown — it never becomes blue-gray.

Code blocks & syntax

Warm syntax tokens — never the bright VSCode default palette. Inline code earns just bg + radius, no syntax colour.

  return (won / calls.length) * 100;

Markdown rendering

Assistant-rendered prose: serif body 16/26, flush on canvas, never inside a card. H1 and H2 stay serif; H3 drops to sans 600.

Heading one

The quietest interfaces feel like reading a book. Body text wraps at a comfortable measure, a serif for the running prose, and sans for everything that asks the user to act. The eye knows where it is at every moment.

Heading two

Inline tokens like const sit on a tinted background; long-form citations link out — see the canonical reference for the full set of rules.

  • Hairline dividers, not thick rules
  • 4px line gaps inside lists
    • Nested level uses an open circle marker
  • List markers in tertiary text
  1. Token-first styling for theme parity
  2. Density is a layer, not a fork
  3. Reduced motion is honoured everywhere

Heading three

Beware of using two affordances for the same job — the eye reads it as redundancy and stops trusting either.

--text-secondarySubtitles, helper text, descriptions
--text-tertiaryMeta, captions, placeholder hints

Streaming text

Per-token, ~30ms cadence, with a sine ease-in on the first 200ms. Punctuation gets a one-tick pause for natural breath. Reduced motion shows the full text immediately.
